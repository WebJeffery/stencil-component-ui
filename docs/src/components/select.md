# Select 下拉选择器

## 常规用法

<swc-select>
    <swc-option value="1" label="苹果"></swc-option>
    <swc-option value="2" label="香蕉"></swc-option>
    <swc-option value="3" label="橘子"></swc-option>
    <swc-option value="4" label="芒果"></swc-option>
</swc-select>

```html
<swc-select>
    <swc-option value="1" label="苹果"></swc-option>
    <swc-option value="2" label="香蕉"></swc-option>
    <swc-option value="3" label="橘子"></swc-option>
    <swc-option value="4" label="芒果"></swc-option>
</swc-select>
```

## 禁用选项

通过给 `select` 设置 `disabled` 属性来禁用下拉框

<swc-select disabled>
    <swc-option value="1" label="苹果"></swc-option>
    <swc-option value="2" label="香蕉"></swc-option>
    <swc-option value="3" label="橘子"></swc-option>
    <swc-option value="4" label="芒果"></swc-option>
</swc-select>
