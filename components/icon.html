安装 ​

现在 swc-icon 被抽离成了一个单独的 npm 包，现在使用时需要单独安装 @swc-ui/icons

bash
npm i @swc-ui/icons

使用 ​

js
// 全量引入
import { registerComponent} from '@swc-ui/icons'
registerComponent()

// 按需引入
import '@swc-ui/icons/dist/components/swc-icon-add-location.js'
